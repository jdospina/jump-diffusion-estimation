Installation
============

.. code-block:: bash

   git clone https://github.com/jdospina/jump-diffusion-estimation.git
   cd jump-diffusion-estimation
   pip install -e .

For development (tests, linting, docs):

.. code-block:: bash

   pip install -e .[dev]

For the tutorial notebooks:

.. code-block:: bash

   pip install -e .[tutorials]
